Política de Privacidade — Lúdica

Última atualização: 14 de julho de 2026

Esta política explica como o Lúdica — sistema de planejamento de sessões com jogos, voltado a profissionais que atendem crianças e adolescentes — coleta, usa e protege dados pessoais. Ela vale para quem cria uma conta e usa a ferramenta (a "profissional") e para os dados de pacientes que a profissional cadastra no sistema.

1. Quem é o responsável pelos dados

O Lúdica é desenvolvido e mantido por Vinícius, responsável técnico pela aplicação. Cada profissional que usa o sistema é responsável pelos dados de seus próprios pacientes que insere na ferramenta, atuando como controladora desses dados perante a Lei Geral de Proteção de Dados (LGPD — Lei nº 13.709/2018). O responsável técnico atua como operador, processando os dados apenas para fazer o sistema funcionar.

Dúvidas ou solicitações sobre dados: contato@seudominio.com

2. Dados que coletamos

Da profissional (titular da conta):

  • Nome e e-mail — informados no cadastro ou obtidos do Google, caso o login seja feito com "Entrar com o Google".
  • Foto de perfil do Google, quando aplicável.
  • Caso a profissional conecte a Google Agenda, um token de acesso temporário é usado para ler e criar eventos — esse token fica apenas no navegador da profissional, não é armazenado em nossos servidores.

Dos pacientes (inseridos pela profissional):

  • Iniciais (não o nome completo), idade, demandas/áreas de desenvolvimento trabalhadas e observações clínicas.
  • Esses dados são considerados dados pessoais sensíveis (dados de saúde, art. 5º, II da LGPD), tratados sob a base legal de tutela da saúde e execução de serviço, conduzidos exclusivamente pela profissional responsável pelo atendimento.

3. Como os dados são usados

  • Organizar o planejamento de sessões, histórico de atendimentos e sugestões de jogos.
  • Quando a profissional usa a "Sugestão inteligente", enviamos à IA (Google Gemini) apenas idade, demanda e observações do paciente — o nome ou iniciais do paciente nunca são enviados ao modelo de IA.
  • Quando a Google Agenda está conectada, o sistema lê e cria eventos diretamente na agenda do Google da profissional, usando a autorização que ela concedeu.
  • Não usamos os dados para publicidade, não vendemos dados a terceiros e não fazemos qualquer uso além do necessário para o funcionamento do Lúdica.

4. Onde os dados ficam armazenados

Os dados de conta, pacientes e sessões ficam armazenados no Supabase, com controle de acesso por linha (Row Level Security): cada profissional só consegue ver e alterar os dados que ela mesma cadastrou. A comunicação com o servidor é criptografada (HTTPS).

5. Compartilhamento com terceiros

  • Google — para autenticação (login) e, se a profissional autorizar, para leitura/criação de eventos na Google Agenda.
  • Google Gemini (IA) — para gerar sugestões de jogos, recebendo apenas idade, demanda e observações (sem identificação do paciente).
  • Supabase — provedor de banco de dados que armazena as informações da conta e dos pacientes.

Nenhum desses terceiros recebe dados para fins próprios de publicidade ou revenda.

6. Backup e exportação

A profissional pode exportar uma cópia de seus dados (pacientes, jogos e sessões) em um arquivo .json a qualquer momento, pela tela de Configurações.

7. Seus direitos como titular de dados

Conforme a LGPD, você pode solicitar a qualquer momento:

  • Confirmação da existência de tratamento e acesso aos dados;
  • Correção de dados incompletos, inexatos ou desatualizados;
  • Exclusão dos dados pessoais (conta e/ou dados de pacientes), exceto quando houver obrigação legal de retenção;
  • Portabilidade dos dados a outro fornecedor, mediante requisição expressa;
  • Revogação do consentimento, incluindo desconectar o acesso à Google Agenda a qualquer momento pela tela de Agenda.

Para exercer esses direitos, entre em contato pelo e-mail acima. Pedidos relacionados a dados de pacientes devem ser direcionados à profissional responsável pelo atendimento, já que ela é a controladora desses dados.

8. Retenção e exclusão

Os dados ficam armazenados enquanto a conta estiver ativa. Ao solicitar o encerramento da conta, os dados são removidos do banco de dados, salvo quando a legislação aplicável (ex: normas de prontuário/registro profissional) exigir guarda por prazo determinado — nesse caso, a responsabilidade de retenção é da profissional, não da plataforma.

9. Cookies e armazenamento local

O Lúdica usa apenas o armazenamento necessário para manter a sessão de login ativa no navegador (via Supabase Auth). Não utilizamos cookies de rastreamento ou publicidade.

10. Alterações nesta política

Esta política pode ser atualizada conforme o sistema evolui. A data no topo desta página indica a versão mais recente. Mudanças relevantes serão comunicadas às profissionais cadastradas.
